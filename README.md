# Shalom Finance - Soluções Financeiras

Plataforma digital completa focada em oferecer soluções financeiras modernas, seguras e transparentes para **Igrejas, OSCs, Pequenos Negócios e Empreendedores**.

## 🚀 Sobre o Projeto

A Shalom Finance atua como uma plataforma de conexão financeira, facilitando a gestão, recebimentos e operações bancárias de forma simplificada e eficiente. O projeto foi desenvolvido com foco em alta performance, usabilidade responsiva e otimização para mecanismos de busca (SEO).

## 🛠 Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web modernas para garantir rapidez e compatibilidade:

* **HTML5 / CSS3:** Estruturação semântica e estilização customizada.
* **Bootstrap 5:** Framework para design responsivo e componentes dinâmicos.
* **Bootstrap Icons:** Biblioteca de ícones vetoriais.
* **JavaScript (Vanilla):** Lógica para gerenciamento de estado (cookies), carregamento (preloader) e navegação suave (*smooth scroll*).

## ✨ Principais Funcionalidades

* **Navegação Fluida:** *Smooth scroll* inteligente com ajuste automático para barra de navegação fixa.
* **Design Premium:** Paleta de cores exclusiva (`#D4AF37`) focada em transmitir confiança e autoridade.
* **Responsividade:** Layout adaptável para desktop, tablets e dispositivos móveis.
* **SEO Otimizado:** Implementação de meta-tags, Open Graph, Twitter Cards e dados estruturados (JSON-LD) para melhor indexação pelo Google.
* **Experiência de Usuário:** Preloader personalizado e banner de cookies funcional.

##📈 SEO e Estrutura
O site foi desenvolvido pensando em visibilidade e indexação:

Canonical Tags: Evita conteúdo duplicado nos mecanismos de busca.

Dados Estruturados: Schema.org integrado (FinancialService) para que o Google reconheça a natureza do serviço financeiro (conta digital, gestão de pagamentos, etc).
